## mongodb

### 环境变量配置

`配置完环境变量可直接在dos窗口通过命令启动 `

>C:\Program Files\mongodb-win32-x86_64-windows-5.0.17\bin

### 启动命令

> 服务端启动命令：mongod	
>
> 客户端启动命令：mongo

### 文档相关命令

1.插入文档

```db
db.集合.insert(文档对象)
```

2.查询文档

```db
db.集合.find(查询条件)
```

3.更新文档

```db
db.集合.update(查询条件,新的文档)
db.集合.update({name:'张三'},{$set:{age:19}})
```

4.删除文档

```db
db.集合.remove(查询条件)
```

### 应用场景

>- 新增
>- 删除 (伪删除)
>- 更新
>- 查询
